Accessibility Widget – Demo

Use the blue button in the lower-right corner to open the settings panel and explore every feature.

Feature Overview

Adjustable via Sliders

  • Font size: 80 – 150 %
  • Line height: 100 – 200 %
  • Letter spacing: 100 – 150 %
  • Content scale: 80 – 130 %
  • Column width: 400 – 1200 px

Instant-Toggle Functions

  • Reading mask
  • High-contrast mode
  • Dark mode
  • Dyslexia-friendly font
  • Text alignment (left, centre, right)
  • Underline links
  • Hide images
  • Highlight headings
  • Text magnifier (cursor reader)
  • Show focus outline
  • Reduce motion
  • Large cursor
  • Stop media autoplay
  • Invert colours

All settings are stored locally in localStorage, so user preferences persist across visits.
